# Instruções para Assistente de IA - Engenharia NATA

Este arquivo é uma referência rápida. Para instruções completas, consulte `.github/copilot-instructions.md`.

## 📋 Visão Geral

**Engenharia NATA** é um portfólio de aplicativos web educativos para engenharia e finanças:
- ✅ 100% JavaScript Puro (sem frameworks)
- ✅ Bilíngue (PT-BR/IT-IT)
- ✅ Mobile-First
- ✅ Completamente comentado em português
- ✅ Inclui memoriais de cálculo educativos

## 🎯 Regras Essenciais

1. **SEMPRE** consulte `.github/copilot-instructions.md` antes de fazer alterações
2. **SEMPRE** use funções de `assets/js/site-config.js` para formatação
3. **SEMPRE** implemente suporte bilíngue (PT-BR/IT-IT)
4. **SEMPRE** comente código em português
5. **NUNCA** use frameworks JavaScript
6. **NUNCA** deixe `console.log` em produção

## 📁 Estrutura de Cada App

Cada app tem 3 arquivos:
- `app-name.html` - Estrutura HTML
- `app-name-script.js` - Lógica JavaScript
- `app-name-styles.css` - Estilos CSS

## 🔧 Funções Importantes (site-config.js)

```javascript
formatarNumero(valor, casasDecimais)
formatarMoeda(valor, idioma)
obterIdiomaAtual() // 'pt-BR' ou 'it-IT'
converterValorFormatadoParaNumero(valorFormatado)
```
