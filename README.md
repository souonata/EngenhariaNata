# Engenharia NATA — Portfólio de Apps Web

Uma pequena coleção de apps web para estudar conceitos práticos:

- Mutuo — calculadora de empréstimos (SAC, Price, Americano) com gráficos simples.
- Helice — calculadora de passo de hélice para barcos de lazer.
- Solar — dimensionamento fotovoltaico off-grid (painéis, baterias, inversor) com página de configuração.

Tudo em HTML, CSS e JavaScript puro — fácil de abrir e usar localmente.

Como usar (rápido):

1. Abra `index.html` no seu navegador (clique duas vezes) ou use Live Server no VS Code.
2. Escolha o app que quer testar.
3. A maioria das páginas é bilíngue: troque o idioma no seletor (Português / Italiano).

Configurações do Solar:

- A página `solar/config.html` permite ajustar preços e especificações (potência de painel, tensão, kWh/Ah, preço e peso).
- As configurações são salvas em `localStorage` sob a chave `configSolar`.

Um glossário simples está disponível no arquivo `GLOSSARIO.md` para explicar termos úteis (kWh, painel, inversor, etc.).
